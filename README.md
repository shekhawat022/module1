📂 Module 1 – Data Science: Introduction to Python
This repository contains all projects and exercises from Module 1 of the Data Science course. The focus is on Python fundamentals, Object‑Oriented Programming, file handling, and introductory work with NumPy and Pandas.

📌 Key Learnings
Core Python programming basics

Object‑Oriented Programming (OOP): classes, objects, methods, encapsulation

File handling: reading, writing, and managing text files safely

NumPy essentials: arrays, slicing, reshaping, arithmetic operations

Pandas basics: DataFrames, Series, statistics, and data cleaning

🖥️ Projects Overview
1. Coffee Machine (OOP Project)
Designed a Coffee Machine simulation using OOP principles

Implemented classes for CoffeeMachine, Coffee, and Ingredient

Features: making coffee, checking resources, handling shortages

2. File Handling Project
Built a file processing application

Reads data from text files, writes processed output to new files

Includes line counting, filtering, and logging

3. NumPy Mini Projects (x5)
Demonstrated array creation and manipulation

Performed element‑wise operations, reshaping, transposing

Applied statistical functions (sum, mean, max, min)

Practiced slicing and boolean indexing

4. Pandas Mini Projects (x5)
Created and explored DataFrames and Series

Loaded datasets from CSV/Excel

Filtered rows/columns with conditions

Computed summary statistics (mean, median, count)

Performed basic cleaning (missing values, duplicates, transformations)
